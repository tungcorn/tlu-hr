# Hồ sơ 360 - Employee Master Detail View

**User Goal:** View and manage comprehensive information about a single staff member across all HR domains in one unified interface.

**Role:** HR Specialist (full access), Department Head (unit staff only), Employee (own profile, limited fields)

## Interaction Rules

### Tab Navigation
| Action | Result |
|--------|--------|
| Click tab | Switch content, preserve scroll position |
| URL reflects tab | `/employee/TLU-2015-0234?tab=chuyen-mon` |
| Browser back | Return to previous tab |

### Edit Mode
| Action | Trigger | Result |
|--------|---------|--------|
| Enter edit mode | Click [Chinh sua] | Form fields become editable |
| Save changes | Click [Luu] | Validate, save, show success toast |
| Cancel | Click [Huy] | Discard changes, confirm if dirty |
| Auto-save | Timeout 30s | Save draft (for long forms) |

### Data Expansion
| Element | Action | Result |
|---------|--------|--------|
| "Xem tat ca" link | Click | Navigate to full list view |
| Table row | Click | Expand inline detail or open modal |
| Timeline segment | Click | Show contract detail modal |

| Field Group | HR Full | Dept Head | Self | Finance |

## Responsive Behavior

### Desktop (> 1024px)
- Full tabbed interface
- Side-by-side columns in content
- Inline editing

### Tablet (768px - 1024px)
- Tabs become scrollable
- Single column layout
- Modal editing

### Mobile (< 768px)
- Tabs become accordion
- Card-based layout
- Full-screen editing forms
